# Pedidos de Reventa PWA

App móvil para productos, pedidos, depósitos, entregas y ganancias.

## Cómo correrla
```bash
npm install
npm run dev
```

## Cómo publicarla en Vercel
1. Sube esta carpeta a GitHub.
2. Entra a Vercel.
3. New Project.
4. Importa el repo.
5. Framework: Vite.
6. Build command: npm run build.
7. Output directory: dist.
8. Deploy.

## Cómo instalarla en celular
Abre el link de Vercel en Safari/Chrome:
- iPhone: Share > Add to Home Screen.
- Android: menú de Chrome > Install app.
